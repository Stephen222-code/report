2. 零食折扣店渠道的结构性机遇

📈
原文要点
  • 中国零食饮料零售市场规模约 Rmb4 万亿（2025E）
  • 折扣店渠道 2019-25E 销售额 CAGR 76%，2025-29E 预计 CAGR 23%，市场份额由 5% 升至 10%
  • 2025E 渠道销售额 Rmb210bn，2029E 达 Rmb485bn
  • 当前门店数约 56,000 家，潜在天花板约 80,000 家，中期仍有超 40% 增长空间
  • 成功关键：简化供应链（生产商→折扣店直采）、高SKU多样性、快速周转、价格优势

深度解读
  • 中国食品制造产能过剩 + 高效低成本物流 + 人口稠密带来的高周转率，共同构筑了折扣店的“经济基础”
  • 传统超市/杂货店份额从 49% 降至 43%（2029E），折扣店正成为消费降级时代最匹配的零售形态
  • 品牌端：零食龙头（如卫龙、洽洽）在折扣店渠道收入贡献已达 25-30%，未来或成为最重要增量渠道
  • 宏观背景：居民消费意愿低迷下，“性价比”成为核心竞争力，折扣店模式无补贴驱动，具备可持续性
  • 风险：若宏观复苏超预期，高端渠道可能回流，但中长期趋势难以逆转

3. Busy Ming 的领导地位与竞争格局

🏆
原文要点
  • 2023年11月 Busy For You 与 ZhaoYiMing 合并成立 Busy Ming Group
  • 截至2025年9月底，门店数 19,517 家，覆盖 28 个省份、1,341 个县
  • 2024年 GMV 超 Rmb55.5bn，是中国零食饮料连锁零售商 No.1，F&B 连锁零售商 No.4
  • 与 Wanchen（好想来）形成双寡头格局，两家合计占行业门店约 2/3
  • 两品牌区域互补：Busy For You 主攻华中，Super Ming 主攻北方
🛡️
深度解读
  • 规模优势已转化为极强议价权，成为众多零食品牌最重要的增量渠道
  • 区域互补策略避免内耗，全国化布局为后续继续整合中小区域玩家提供基础
  • 私标产品（2025年起推出）将进一步提升差异化与毛利率，类似美国 Aldi 或中国山姆会员店模式
  • 行业集中度加速提升，小型区域玩家市占率均低于 10%，龙头整合窗口期已开启

4. 门店扩张与同店销售稳定

📍
原文要点
  • 2025-27E 年末门店数：21,000 / 24,000 / 27,000 家（YoY +47% / +15% / +11%）
  • 59% 门店位于县城及乡镇，深度下沉至低线市场
  • 单店日均销售额预计 2025-27E 分别 -4% / -3% / -2%（密度上升但 snacks+ 策略对冲）
  • 门店模型演进：1.0（150㎡，1800+ SKU）→ 2.0 大店 → 3.0 折扣超市（新增日用品）
📊
深度解读
  • 下沉市场仍是最大蓝海，县域渗透率远低于一二线，扩张空间充足
  • snacks+（饮料、冷冻品、私标）策略有望稳定客流与客单价，抵消密度上升带来的同店压力
  • 单店回本期从早期 1 年延长至当前 2.5 年，成熟期预计 3-3.5 年，仍优于多数加盟模式
  • 未来向折扣超市转型将打开更广阔的 FMCG 市场，长期天花板远超当前零食品类

5. 产品策略、供应链优化与利润率提升

📦
原文要点
  • 毛利率 2025-27E 预计 +2.2 / +0.3 / +0.3ppt 至 9.8% / 10.1% / 10.4%
  • SG&A 费用率 +0.3 / -0.2 / -0.1ppt 至 5.1% / 4.9% / 4.8%
  • 经常性净利率 2025-27E 提升至 3.9% / 4.2% / 4.3%
  • 私标产品已于 2025 年推出，价格较竞品低 28% 左右
  • 产品品类从零食向 snacks+ 扩张，饮料贡献从 21% 升至 27%（9M25）
📉
深度解读
  • 规模效应 + 直采模式使渠道加价率仅 36%（KA 为 55%），毛利率提升空间清晰
  • 私标占比提升将显著改善产品组合，成为长期毛利率扩张的核心引擎
  • 运营杠杆显现：固定成本随门店扩张被快速摊薄，SG&A 费用率下降趋势明确
  • 供应链优化不仅带来价格优势，更增强了与品牌方的战略合作深度，形成正向循环

7. 估值分析与情景模拟

📉
原文要点
  • 目标价 HK$500 对应 2026E PE 29x（23% 净利润 CAGR）
  • 当前交易于 22x/20x 2026/27E PE，高于零食同业（19x/16x），低于部分零售同业
  • 上行情景（HK$550）：收入 CAGR 22%，GM 额外 +80bps

⚖️
深度解读

  • 上行情景若实现，2027E 净利润或超预期 15-20%，目标价具备进一步上修空间
  • 下行风险主要来自宏观超预期疲软或门店扩张放缓，但行业天花板支撑长期韧性
  • 与 Miniso、Wanchen 等可比公司对比，Busy Ming 的增长确定性与稀缺性更强
